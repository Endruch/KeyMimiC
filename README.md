This program is designed for full automation of keyboard and mouse actions using customizable macros.
It supports two independent execution threads, allowing you to run two parallel macros at the same time — useful for gaming scenarios or repetitive multitasking workflows.

You can create sequences of virtual key presses, mouse movements, clicks, delays, and loops, making the tool flexible for many automation needs.
